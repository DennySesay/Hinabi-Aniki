# Hinabi Aniki Configuration

- **deckSkillMap**: A mapping of your exact Anki deck names to one of the 6 skills reognized by Hinabi (`vocabulary`, `grammar`, `kanji`, `reading`, `listening`, `speaking`). For subdecks, use Anki's standard notation (e.g., `Japanese::Grammar`).
- **sessionTimeoutMinutes**: The number of minutes of inactivity before a session is automatically ended.
- **hinabiEndpoint**: The URL of your Hinabi instance's API endpoint for Anki sessions.
- **apiKey**: Your Hinabi API key for authenticating the request.
